`league/squery` is a neat wrapper for [osquery](http://osquery.io). It provides a fluent interface for building SQL queries against system-level data.

## Installation

Install `league/squery` through composer.

~~~ json
{
    "require": {
        "league/squery": "~2.0.0"
    }
}
~~~
